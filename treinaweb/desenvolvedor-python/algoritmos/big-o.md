# Análise temporal

## Busca Linear

- Melhor caso: 1 vez
- Pior caso: N vezes
- Caso Médio: N/2 vezes

## Selection Sort

- Melhor caso: N \* N = N^2
- Pior caso: N \* N = N^2
- Caso Médio: N \* N = N^2

# Big-O

## Busca Linear

- Melhor caso: 1 vez | O(1)
- Pior caso: N vezes | O(N)
- Caso Médio: N/2 vezes

## Selection Sort

- Melhor caso: N \* N = N^2 | O(N^2)
- Pior caso: N \* N = N^2 | O(N^2)
- Caso Médio: N \* N = N^2
